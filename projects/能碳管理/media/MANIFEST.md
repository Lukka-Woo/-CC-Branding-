# media/ 确认素材清单

> **AI 必读。** 每次生成任何格式的文档前，必须读取本文件。
> 所有在此登记的素材，生成时**必须**出现在对应位置。
> 每次向 `media/` 添加新文件后，人工在此补充登记。

---

## 使用规则

| 规则 | 说明 |
|---|---|
| **必须使用** | 所有已登记素材，生成时必须出现在输出中 |
| **位置优先** | `用于` 字段明确时，按字段放置；不明确时，放在视觉上最合适的位置 |
| **格式适配** | `适用格式` 为空时，适用所有格式（HTML / PDF / PPTX / DOCX） |
| **禁止裁剪核心内容** | 图片可等比缩放，但不可裁剪掉主体 |
